## Project

Write an app that displays a list of videos

### Requirements

- See the videos listed in a directory
- Ability to like / dislike a video
- Ability to search through videos

- The project should be runnable on any machine and come with instructions on how to start it

### Details

- Use any framework/library you are most comfortable with or none at all
- The time allocated should be no longer than 2 hours

### Optional

- Write unit tests for your code
